Alula Plugins

Alula Plugins is a collection of simple and useful plugins that extends functionality of Alula.

Plugins

Lightbox

Lightbox displays your photos beautifully using FancyBox. 

{% lightbox "image" ["title"] ["alt"] %}
  • image is the image name in attachments/_originals directory.
  • title is used to display caption.
  • alt is alternative text for image if it cannot be displayed.
